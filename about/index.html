Why Canvas on Tap exists

Hi, I’m Will.

Canvas on Tap is my project for making Canvas automation approachable. I build practical scripts and tools using Scriptable on iPad so everyday workflows in Canvas are faster, safer, and easier to repeat.

My goal: help non-programmers feel confident using small scripts to solve real problems.

What you’ll find here

  • Blog posts that explain workflows step-by-step
  • Tools that you can copy/paste and run right away
  • Patterns for building safer scripts (Keychain, logging, pagination)

The “Tap” mindset

Most automation doesn’t need to be complicated. A good script is often just a repeatable pattern: gather input → make a request → process results → output something useful.
